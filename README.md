# pokedexcli
Pokedex CLI tool for practicing working with JSON data and making requests to web servers. A BootDev course.

Follows the typical CLI REPL program tool. There are a number of commands that allows
a user to navigate through a pokemon world using the PokeApi provided at [PokeApi](https://pokeapi.co/api/v2/location-area/)

Build Command
`go build`

Run Command
`go run .`

Test Command
`go test ./...`

Type help.... for help!
